## <span style="color:#E91E63; font-weight:700;">Team Charter Appendix A</span>

<span style="color:#E91E63; font-weight:700;">Team Goals for Development</span>

## <span style="color:#E91E63; font-weight:700;">Data Collection & Transmission</span>
<ul style="list-style-type: disc;">
  <li><span style="color:#E91E63;"></span> Ensure the weather station can consistently send real-time data to a central server/cloud.</li>
  <li><span style="color:#E91E63;"></span> Implement redundant communication methods (e.g., LoRa, Wi-Fi, LTE) for reliability in remote areas.</li>
  <li><span style="color:#E91E63;"></span> Include local data logging (SD card or onboard memory) in case of communication failure.</li>
</ul>

<hr style="border: 1px solid #E91E63;">

## <span style="color:#E91E63; font-weight:700;">Environmental Monitoring</span>

<ul style="list-style-type: disc;">
  <li><span style="color:#E91E63;"></span> Measure temperature (ambient + heat spikes).</li>
  <li><span style="color:#E91E63;"></span> Record humidity and wind speed/direction for fire spread prediction.</li>
  <li><span style="color:#E91E63;"></span> Detect air quality indicators such as smoke and particulates (PM2.5, PM10).</li>
</ul>


<hr style="border: 1px solid #E91E63;">

## <span style="color:#E91E63; font-weight:700;">Fire Detection & Prevention</span>

<ul style="list-style-type: disc;">
  <li><span style="color:#E91E63;"></span> Trigger alerts when abnormal heat or smoke is detected.</li>
  <li><span style="color:#E91E63;"></span> Establish threshold levels for different risk stages (warning, alert, emergency).</li>
  <li><span style="color:#E91E63;"></span> Support automated early warning systems (e.g., SMS/email/IoT alerts).</li>
</ul> 

<hr style="border: 1px solid #E91E63;">

## <span style="color:#E91E63; font-weight:700;">Power & Reliability</span>

<ul style="list-style-type: disc;">
  <li><span style="color:#E91E63;"></span> Design a solar + battery backup system for continuous operation.</li>
  <li><span style="color:#E91E63;"></span> Implement low-power modes to conserve energy when idle.</li>
  <li><span style="color:#E91E63;"></span> Ensure rugged, weatherproof housing for harsh outdoor environments.</li>
</ul>


<hr style="border: 1px solid #E91E63;">

## <span style="color:#E91E63; font-weight:700;">Collaboration & Documentation</span>

<ul style="list-style-type: disc;">
  <li><span style="color:#E91E63;"></span> Maintain clear GitHub repository for source code, schematics, and documentation.</li>
</ul>
